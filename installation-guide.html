Check What's Already Installed
Run these first — skip any step where it's already installed
DO THIS FIRST

Open your terminal first:

💡 To open Terminal on Mac: Press ⌘ + Space, type Terminal, press Enter.
💡 To open Command Prompt on Windows: Press Windows key, type cmd, press Enter. Or search for Command Prompt in Start menu.

Now paste each command and check the result:

CHECK · Terminal / Command Prompt
node --version # Check Node.js git --version # Check Git claude --version # Check Claude Code

What the output means:

✓ Already installed
You see a version number like v20.11.0 or git version 2.39.0 — skip that step below.
✗ Not installed
You see command not found or 'node' is not recognized — follow that step below.

Install Claude Code
AI assistant that runs on your desktop — automates everything
RECOMMENDED
🤖 What Claude Code does for you: Instead of typing terminal commands yourself, you just describe what you want in plain English — "push my project to GitHub" or "run the backtester on HDFCBANK" — and Claude Code does it. Requires Node.js (Step 1) to be installed first.
  • 1
    Install Claude Code globally
    Run this single command in your terminal:
    TERMINAL / COMMAND PROMPT
    npm install -g @anthropic-ai/claude-code
    ⏳This takes 1–3 minutes. Wait for it to finish completely before the next step.
  • 2
    Verify the installation
    TERMINAL / COMMAND PROMPT
    claude --version # Should show a version number
  • 3
    Log in with your Anthropic account
    Navigate to your project folder and start Claude Code:
    TERMINAL / COMMAND PROMPT
    cd india-trading-system # go into your project folder claude # starts Claude Code
    It will open a browser to log in with your Anthropic/Claude account. Once logged in, come back to the terminal — you can now talk to it in plain English.
    ✅You're now set up. Try typing: "push this project to my GitHub account achallakiran"
4
Create GitHub Repository
One-time setup on github.com — done in your browser
REQUIRED
  • 1
    Create the repository
    Go to github.com → log in as achallakiran → click the + icon (top right) → New repository
    Repository name:india-trading-system
    Visibility:● Public (required for free GitHub Pages)
    Initialize:⚠ Leave ALL checkboxes empty — do NOT add README, .gitignore, or license
  • 2
    Enable GitHub Pages
    In your new repo → click Settings tab → in the left sidebar click Pages → under Source select GitHub Actions → click Save
    ℹ️This is what makes your site auto-deploy every time you push code. You only do this once.
  • 3
    Create a Personal Access Token (your Git password)
    GitHub no longer accepts your account password in the terminal. You need a token:
    1. GitHub → click your profile photo (top right) → Settings
    2. Scroll down the left sidebar → click Developer settings
    3. Click Personal access tokens → Tokens (classic)
    4. Click Generate new token (classic)
    5. Note: write india-trading — Expiration: No expiration
    6. Tick the repo checkbox (first one in the list)
    7. Scroll down → click Generate token
    8. Copy the token immediately — GitHub shows it only once
    ⚠️Save this token somewhere safe (like Notes or a password manager). You'll need it the first time you push. Treat it like a password — never share it.
5
Deploy to GitHub Pages
Push your code — site goes live automatically
REQUIRED
  • 1
    Unzip the project
    Download the ZIP from our chat conversation. Unzip it. You'll get a folder called india-trading-system.
  • 2
    Open terminal inside the folder
    Open Terminal. Type cd (with a space after), then drag the folder from Finder into the Terminal window. Press Enter.
    Open the india-trading-system folder in File Explorer. Click the address bar at the top (where it shows the folder path). Type cmd and press Enter. A Command Prompt opens inside that folder.
  • 3
    Run these commands — one by one
    Copy and paste each line, press Enter, wait for it to finish before the next:
    TERMINAL / COMMAND PROMPT · Run in order
    git init git add . git commit -m "Initial: India Trading System v1.0" git branch -M main git remote add origin https://github.com/achallakiran/india-trading-system.git git push -u origin main
    🔑When terminal asks for Username: type achallakiran
    When it asks for Password: paste your Personal Access Token from Step 4 (not your GitHub password). The token won't show as you type — that's normal.
  • 4
    Wait 2 minutes — then open your site
    GitHub Actions is building your site automatically. After ~2 minutes, open:
    YOUR LIVE URL
    https://achallakiran.github.io/india-trading-system
    🎉Your India Trading System is live! Bookmark this URL.
